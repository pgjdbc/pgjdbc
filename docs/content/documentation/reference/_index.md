The complete catalogue of every connection property the driver
recognises. Pages here are generated from annotations on the driver
source code (`@PgApi`, `@PgTags`, `@PgPropertyType`) and regenerated on every
build, so they cannot drift from the running driver.

For topical guidance see the other sections; come here when you need
the full set of tunables, defaults, types, and version each was
introduced.
