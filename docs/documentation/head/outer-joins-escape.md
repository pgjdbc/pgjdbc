You can specify outer joins using the following syntax: `{oj table (LEFT|RIGHT|FULL) OUTER JOIN (table | outer-join)
ON search-condition  }`

For example :

```java
rs = stmt.executeQuery( "select * from {oj a left outer join b on (a.i=b.i)} ");
```
