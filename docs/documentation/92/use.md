<a name="import"></a>
# Importing JDBC

Any source that uses JDBC needs to import the `java.sql` package, using:

import java.sql.*;

### Note

You should not import the `org.postgresql` package unless you are not using standard
PostgreSQL™ extensions to the JDBC API.
